# Student Performance Analysis
## Overview
This project analyzes the relationship between study hours and student exam performance using Python.

## Objectives
- Perform exploratory data analysis
- Calculate statistical metrics
- Visualize study patterns
- Understand correlation between study time and exam results

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib

## Dataset
The dataset contains information on study hours and exam scores for students.

## Project Structure
analysis.ipynb – Python notebook containing analysis code  
student_scores.csv – dataset used in the project

## Key Insight
The analysis shows a positive relationship between study hours and exam performance.
